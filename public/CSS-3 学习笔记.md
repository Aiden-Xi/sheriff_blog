1. CSS3 边框<br>
边框有如下相关的属性： border-radius(圆角), box-shadow(盒阴影), border-image(边界图片).

2. CSS 3 圆角<br>

3.  CSS 3 背景<br>
背景相关属性：
    - background-image
    - background-size
    - background-origin
    - background-clip

4. CSS 3 渐变<br>
   渐变相关的属性有两个：
   - linear-gradients: 线性渐变（向下/向上/向左/向右/对角方向）
   - radial-gradients: 径向渐变（由中心d定义）

5. CSS 3 文本效果<br>
   文本效果相关属性值：
   - text-shadow
   - box-shadow
   - text-overflow
   - word-wrap
   - word-break

6. CSS 3 字体<br>
   字体所对应的基本属性：   
   - font-face
   ```
   实例：
    <style> 
        @font-face
        {
            font-family: myFirstFont;
            src: url(sansation_light.woff);
        }

        div
        {
            font-family:myFirstFont;
        }
    </style>
   ```
